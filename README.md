# react-native-pdf417
React-Native library which allows you to generate a barcode in pdf417 format
## Installation

```sh
npm install react-native-pdf417
```

## Usage

```js
import { Pdf417View } from "react-native-pdf417";

// ...

<Pdf417View color="tomato" />
```
